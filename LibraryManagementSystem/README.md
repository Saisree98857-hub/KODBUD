# Library Management System

A simple console-based Library Management System built using Java.  
This project allows users to manage books using basic CRUD operations.

---

## Features

- Add New Books
- Display All Books
- Search Book by Title
- Remove Book
- Menu-Driven Console Application
- Uses ArrayList and Custom Classes

---

## Technologies Used

- Java
- VS Code
- ArrayList
- OOP Concepts
