# 📚 Library Management System

A simple and beginner-friendly **Library Management System** built using **Java**.  
This is a **console-based application** that helps users manage books efficiently using basic CRUD operations.

The project is developed as part of a Java programming task and demonstrates the use of:

- Java Classes & Objects
- ArrayList
- Methods
- Loops
- Conditional Statements
- CRUD Operations
- Menu-Driven Programming

---

# 🚀 Features

✅ Add New Books  
✅ Display All Books  
✅ Search Book by Title  
✅ Remove Books  
✅ Menu-Driven Console Interface  
✅ Uses Java ArrayList  
✅ Beginner Friendly Project  
✅ Clean and Simple Code Structure

# 📖 How the Program Works

The system displays a menu with multiple options:

```text
===== LIBRARY MANAGEMENT SYSTEM =====
1. Add Book
2. Display All Books
3. Search Book
4. Remove Book
5. Exit
```

Users can choose options by entering numbers.

---

# ▶️ How to Run the Project

## Step 1: Open Project Folder in VS Code

Open the folder containing:

```text
LibraryManagementSystem.java
```

---

## Step 2: Compile the Java Program

Open terminal in VS Code and run:

```bash
javac LibraryManagementSystem.java
```

If there are no errors, compilation is successful.

---

## Step 3: Run the Program

```bash
java LibraryManagementSystem
```
