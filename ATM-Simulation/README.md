# Java ATM Simulation

This is a simple ATM Simulation project developed using Java.

## Features
- Check Balance
- Deposit Money
- Withdraw Money
- Exit Option

## Technologies Used
- Java
- VS Code

## Concepts Used
- Loops
- Methods
- Switch Case
- Conditions
- User Input

## How to Run
1. Open project in VS Code
2. Run `ATMSimulation.java`
3. Use the ATM menu options
