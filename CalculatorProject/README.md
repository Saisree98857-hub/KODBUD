# Simple Calculator in Java

A simple CLI-based calculator application developed using Java as part of the Kodbud Java Programming Internship.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- User-friendly console interaction
- Menu-driven operations

## Technologies Used

- Java
- VS Code
- Git & GitHub

## Concepts Practiced

- Variables and Data Types
- Scanner Class
- switch-case
- User Input
- Arithmetic Operations

## How to Run

Compile the program:

javac Calculator.java

Run the program:

java Calculator

## Sample Output

===== Simple Calculator =====

Enter first number: 10
Enter operator (+, -, *, /): +
Enter second number: 5

Result = 15
